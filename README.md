# ant-design-pro-vue

## 源起

element-ui 是饿了么出品的 ui 框架,它为 Vue 开发者提供了丰富的组件库，但是在 UI 风格上更偏活泼，缺少统一标准，Ant Design 作为一门设计语言，经历过多年的迭代和积累，它对 UI 的设计思想已经成为一套事实标准，使用 Ant Design Vue，能使开发者更加专注业务功能的开发实现。

## 项目依赖

Ant-design-pro-vue 主要是基于 vue-cli3 模板为基础开发的，引入了如下 dependencies:

- ant-design-vue Ant Design 的 Vue 实现，开发和服务于企业后台产品的 UI 组件库。
- axios 一个现在主流并且很好用的请求库 支持 Promise
- JWT 权限验证支持
- nprogress 轻量的全局进度条控制
- vuex 官方状态管理
- vue-router 官方路由

该项目从工作流出发，提炼出管理后台所需要的基础必备功能和业务痛点，封装了 axios 请求，支持无限层级路由，动态权限和动态侧边栏。
